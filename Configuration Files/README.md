Digital Treasurer

A professional financial management application built with React and Firebase.

Setup Instructions

Install Dependencies:
Open your terminal in this folder and run:

npm install


Start Development Server:

npm run dev


Build for Production:

npm run build


Features

Real-Time Database: Synced across devices using Firestore.

Glassmorphism UI: Modern, translucent design.

Accessibility Mode: High contrast mode for elderly users.

WhatsApp Reports: One-click colorful report generation.

Google Sheets Import: Fetch data from CSV links.
